How it works

Three deterministic stages. No surprises.

Ingest

Receive streams from any source via SRT, SRTLA, RTMP, UDP or RTP. Protocol-agnostic input layer with automatic reconnection.

Process

Optional transcoding, audio matrix routing, format conversion. Zero-copy passthrough when processing isn't needed.

Redistribute

Fan-out to multiple destinations simultaneously. SRT, RTMP push, multicast UDP. One input, unlimited outputs.
